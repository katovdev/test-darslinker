Darslinker Frontend Next
=======================

This is the frontend application for Darslinker built with Next.js.

Getting Started
---------------
To get started with the Darslinker Frontend Next application, follow these steps:
1. Clone the repository:
2. Install dependencies:
```bash
npm install
## or
yarn install
```
3. Run the development server:
```bash
npm run dev
## or
yarn dev
```
4. Open your browser and navigate to `http://localhost:3000` to see the application running.
5. To build the application for production, run:
```bash
npm run build
## or
yarn build
```
6. To start the production server, run:
```bash
npm start
## or
yarn start
```
